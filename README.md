## 下载

### Docker

```
docker pull apachecn0/apachecn-python-docs-zh
docker run -tid -p <port>:80 apachecn0/apachecn-python-docs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-python-docs-zh
apachecn-python-docs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
